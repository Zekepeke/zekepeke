### moldmany – Character-Level Language Model

Bigram-based generative model inspired by Andrej Karpathy's work. Learns character-to-character transition probabilities from text corpora to generate novel sequences. Demonstrates foundational ML concepts: vocabulary building, probability matrices, and statistical language modeling without deep learning frameworks. Repository: [moldmany](https://github.com/Zekepeke/moldmany)
